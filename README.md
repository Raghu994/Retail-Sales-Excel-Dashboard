# Retail Sales Analysis Dashboard 

## Project Overview
This project analyzes retail sales data to identify trends, top-performing categories, and customer buying behaviors. The goal was to transform raw sales data into an interactive dashboard to support business decision-making.

**Tools Used:** Microsoft Excel (Data Cleaning, Pivot Tables, Charts, Slicers)

## Files Description
- **Retail_Sales_Dashboard_Project.xlsx**: The complete workbook containing:
  - `Raw Data`: The original unprocessed dataset.
  - `Cleaned Data`: The processed dataset with fixed missing values and standardized formatting.
  - `Pivot Tables`: The analysis tables.
  - `Dashboard`: The final interactive visual report.
  - `Insights`: Written conclusions derived from the analysis.

## Process Workflow
1.  **Data Cleaning:**
    - Removed duplicate records.
    - Handled null values in `Price` and `Quantity` columns using calculated formulas.
    - Standardized date formats and category names.
2.  **Data Processing:**
    - Used Pivot Tables to summarize data by Category, Timeline, and Location.
3.  **Dashboard Creation:**
    - Built dynamic charts (Bar, Line, and Donut charts).
    - Added **Slicers** to allow filtering by Payment Method.
